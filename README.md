### Title
RuangKampus Frontend

### descriptionn
Aplikasi ini dibangun menggunakan React, Vite, TypeScript, dan Material UI dengan tampilan clean dan minimalis untuk memudahkan pengelolaan data ruangan dan booking.

Frontend ini berfungsi sebagai antarmuka pengguna untuk:

Menampilkan daftar ruangan

Mengajukan booking ruangan

Mengelola status peminjaman (Approve / Reject)

Mengedit dan menghapus data

### Features
Features

Beberapa fitur utama yang tersedia:

CRUD Data Ruangan

CRUD Data Booking

Approve / Reject Booking

Status badge (Pending, Approved, Rejected)

Filter dan pencarian data

Tampilan clean minimalis (campus style)

Responsive layout

Header fixed saat scroll

### tech stack
Project ini menggunakan teknologi berikut:

React

Vite

TypeScript

Material UI (MUI)

Axios

React Router

### installation
1. Clone Repository
git clone <link-repository>
cd <nama-folder-project>

2. Install Dependencies
npm install

3. Setup Environment Variable

4. Buat file .env di root folder dan isi dengan:
VITE_API_URL=http://localhost:port/api

### usage
Untuk menjalankan project:
npm run dev

Aplikasi akan berjalan di:
http://localhost:port
